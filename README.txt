Программа Light_Curve аппроксимирует наблюдаемую кривую блеска рядом Фурье до 3-го порядка и отображает фазовую кривую. 

Входные данные передаются программе с помошью ключей командной строки:
-p, --period период изменения величины
-f, --file   название файла с входными данными (и путь к нему)

Файл должен состоять из двух колонок: первая - моменты времени, вторая - наблюдаемые величины.
